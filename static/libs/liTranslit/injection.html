liTranslit - jQuery Translit или транслитерация кириллицы в латиницу

Описание Инъекцию собственных правил транслитерации
Результат

default

reg:'"ё"="yo"'

reg:'"ё"="jooo","ж"="zzzzzz"'

Подключаемые файлы 

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
<script src="js/jquery.liTranslit.js"></script>
Инициализация плагина 

<script>
$(function(){
	$('.demo4_text').liTranslit({
		elAlias: $('.demo4_translit')
	});
	$('.demo4_text').liTranslit({
		elAlias: $('.demo4_translit2'),
		reg:'"ё"="yo"'
	});
	$('.demo4_text').liTranslit({
		elAlias: $('.demo4_translit3'),
		reg:'"ё"="jooo","ж"="zzzzzz"'
	});
});
</script>
Код HTML 

<body>
	<input type="text" class="demo4_text" value="ёжик">
	<input type="text" class="demo4_translit">
	<input type="text" class="demo4_translit2">
	<input type="text" class="demo4_translit3">
</body>
