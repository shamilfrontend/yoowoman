liTranslit - jQuery Translit или транслитерация кириллицы в латиницу

Описание Триггер событий
Результат
Транслитерировать
Подключаемые файлы 

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
<script src="js/jquery.liTranslit.js"></script>
Инициализация плагина 

<script>
$(function(){
	$('.demo7').liTranslit({
		status:false
	});
	$('.trigger').on('click',function(){
		$('.demo7').liTranslit('enable');
		$('.demo7').trigger('start');	
		return false
	});
});
</script>
Код HTML 

<body>
	<input type="text" class="demo7">
	<a class="trigger" href="#">Транслитерировать</a>
</body>
