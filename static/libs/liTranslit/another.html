liTranslit - jQuery Translit или транслитерация кириллицы в латиницу

Описание Транслит текста в другой элемент
Результат

Подключаемые файлы 

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
<script src="js/jquery.liTranslit.js"></script>
Инициализация плагина 

<script>
$(function(){
	$('.demo2_text').liTranslit({
		elAlias: $('.demo2_translit')
	});
});
</script>
Код HTML 

<body>
	<input type="text" class="demo2_text">
	<input type="text" class="demo2_translit">
</body>
